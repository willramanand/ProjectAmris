What is Project Quartz? A framework-agnostic UI component library built with Web Components and Lit. It provides a clean, modern design language with full theming support. How do I customize the theme? Override the CSS custom properties on the theme provider. All primitive token scales use role-based names (primary, secondary, etc.) so you can swap entire palettes. Which frameworks are supported? Quartz works with any framework — plain HTML, React, Vue, Svelte, Angular — because it's built on standard Web Components.
